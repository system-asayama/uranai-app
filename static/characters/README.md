# キャラクター画像の差し替え方法

ここに `<干支番号>.<拡張子>` というファイル名で画像を置くと、
自動生成される SVG アバターの代わりにその画像が使われます。

- 干支番号は 0〜59（0=甲子 … 59=癸亥）。各番号と人物の対応は早見表
  （`/uranai/list`）または `sangokushi.py` を参照してください。
- 対応拡張子（優先順）: `png` → `jpg` → `jpeg` → `webp` → `svg`
- 例: 甲子（劉備, No.1）に画像を付ける → `0.png` を置く
- 正方形・360×360px 程度を推奨（円形に切り抜いて表示されます）

画像が無い番号は、勢力カラーの武将エンブレム風 SVG が自動表示されます。

※実在の人物イラスト等を使う場合は、各画像の著作権・利用条件にご注意ください。
